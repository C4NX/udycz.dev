## About

This is the source code for my personal website, [udycz.dev](https://udycz.dev). It's build with [Vite](https://vitejs.dev) and [React](https://reactjs.org).

## Development

To run the project locally, you need to have [Node.js](https://nodejs.org) installed. Then, run the following commands:

```bash
# Clone the repository
git clone https://github.com/C4NX/udycz.dev
# Enter the project directory
cd udycz.dev
# Install dependencies
npm install
# Start the development server
npm run dev
```
